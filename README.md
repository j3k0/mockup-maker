# Mockup Maker

Generate Balsamiq mockups from text description

## Installation

    npm install -g mockupmaker

## Usage

    Usage: mockupmaker -i file.txt > file.bmml

## Example input file

```
    === welcome ===

    __ logo
    _ enter, (>)

    === login ===

    * email:
    * password:
     , forgot your password? >

    _ login with Facebook, (>)

    _ login as a guest, (>)
```
